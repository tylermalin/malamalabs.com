Company

To care for the physical world, you must first measure it.

Mālama is the Hawaiian word for caring, protecting, tending. We chose it because what we are building is the infrastructure that makes it possible to prove, with cryptographic certainty, that the physical world is being cared for.

01 · Origin

A 30-year data trust crisis.

Every carbon market built since 1997 has eventually collapsed into the same gap: there has never been a system capable of producing verifiably true physical-world data at scale.

The EU Emissions Trading System crashed in 2013 after over-allocation was discovered, with carbon prices dropping ninety percent. In 2023, a Guardian investigation found that more than ninety percent of credits from Verra's VCS rainforest portfolio were phantom. Study after study exposes the same pattern: carbon offsets lack the integrity to justify their use as climate mitigation instruments.

The industry's response to every failure was the same: revise the methodology and add another audit layer. We asked a different question.

What if the sensor itself signed the data?

The founding premise of Mālama Labs

02 · The Answer

Hardware Supremacy.

Replace human trust with mathematical certainty. Move the point of verification from auditors to silicon. To forge data, an actor would have to physically dismantle the chip and break elliptic curve cryptography. The solution is not better auditing — it is removing auditors from the verification equation entirely.

SECURE ENCLAVE

ATECC608B Co-Processor

Every Genesis 300 node carries a dedicated cryptographic co-processor. The private key is permanently provisioned at manufacture. There is no software path to extract it. It remains isolated inside the enclave for the device's entire lifetime.

SIGNED AT SOURCE

ECDSA Signatures

Every data packet is signed using Elliptic Curve Digital Signature Algorithm before it leaves the sensor. The signature binds a reading to a specific device, a specific timestamp, and the device's private key. No valid signature, no valid data.

ON-CHAIN ANCHOR

Cardano Provenance

SaveCards anchored to Cardano via CIP-25/CIP-68. Live on Cardano preprod with 2,786+ records since June 2024. Mainnet deployment follows smart contract audit, targeting Q2 2026.

04 · Platform Logic

Build the input once. License it everywhere.

Cost Structure

Near-zero marginal cost to serve new markets

Once Genesis hardware is deployed, each new market adds licensing revenue without new capital expenditure in sensors. The same node validates carbon SaveCards, AI compute packets, and parametric insurance triggers simultaneously.

Regulatory Advantage

One hardware truth layer satisfies multiple compliance regimes

As standards tighten — EU CSRD, SEC climate disclosure, ISO Sustainable AI, EUDR — the same hardware-signed data satisfies multiple regulatory requirements simultaneously. Compliance becomes a by-product of the infrastructure.

Network Advantage

More markets consuming the same data stream increases network value

Every Genesis node validates across multiple data types. More markets consuming the same stream increases enterprise demand for Hex Node validation and drives protocol revenue growth independent of token price.
